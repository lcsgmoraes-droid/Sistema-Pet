# Roteiros de Criativos de Venda - Sistema Pet

Uso: banco inicial de roteiros para anuncios, Reels, Shorts e criativos de
prospeccao. Cada roteiro pode virar video vertical de 15s a 35s.

## Estrutura padrao

1. Gancho nos primeiros 3 segundos.
2. Dor operacional clara.
3. Tela real do Sistema Pet mostrando a solucao.
4. Beneficio pratico.
5. CTA direto.

## Criativo 1 - Lucro real

| Campo | Conteudo |
|---|---|
| Dor | Vender muito e nao saber se teve lucro |
| Tela base | DRE, relatorio de vendas, dashboard financeiro |
| Duracao | 25s |
| CTA | Veja seu lucro real no Sistema Pet |

Narracao:

```text
Vender muito nao significa lucrar.
Se o custo, o imposto e as despesas ficam espalhados, o lucro vira chute.
No Sistema Pet, voce acompanha vendas, custos, contas e resultado em uma rotina mais clara.
Assim fica mais facil saber o que entrou, o que saiu e o que sobrou.
Sistema Pet: gestao para quem quer enxergar o negocio de verdade.
```

Texto na tela:

```text
Vendeu bem, mas lucrou?
Custos + vendas + financeiro
Resultado mais claro
Veja seu lucro real
```

Prompt para IA:

```text
Crie um video vertical de 25 segundos para vender o Sistema Pet com foco em lucro real.
Use uma linguagem simples para dono de pet shop. A cena principal deve ser gravacao
de tela do DRE e relatorios financeiros. Inclua sugestoes de cortes, zooms em numeros
e legendas curtas. Nao prometa aumento garantido de lucro.
```
