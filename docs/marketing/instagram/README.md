# Instagram CorePet

Este diretório concentra o kit de lançamento do perfil `@corepet.erp`.

## Gerar novamente

```powershell
node scripts/render_corepet_instagram_launch_kit.mjs
```

O comando gera três posts quadrados e cinco capas de destaques em `assets/`.
As legendas e a ordem recomendada estão em `CAPTIONS.md`.

O arquivo `source/corepet-lancamento-gestor-petshop.png` foi criado com a ferramenta
de geração de imagens do Codex para uso exclusivo na peça de lançamento. As demais
peças reutilizam telas e materiais reais já existentes no projeto.

## Tráfego pago

- `PLANO_TRAFEGO_PAGO.md`: fundamentos, medição e segurança dos ativos.
- `CAMPANHA_META_LOJISTAS.md`: configuração detalhada do primeiro teste para donos
  e gestores de lojas pet, incluindo público, anúncios, formulário, orçamento e
  critérios de otimização.
