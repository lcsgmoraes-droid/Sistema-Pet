## Antes da implantacao

- [ ] Cliente recebeu e aceitou o escopo e os itens fora do escopo.
- [ ] Producao esta saudavel e sem incidente P0 aberto.
- [ ] Empresa/tenant foi criada com plano `basico`.
- [ ] E-mail do usuario principal foi confirmado.
- [ ] Nenhuma senha, token ou documento sensivel foi colocado neste checklist.

## Configuracao minima

- [ ] Dados basicos da empresa conferidos.
- [ ] Usuario administrador e usuarios operacionais revisados.
- [ ] Permissoes basicas revisadas.
- [ ] Formas de pagamento necessarias cadastradas.
- [ ] Categorias, marcas e departamentos iniciais cadastrados.
- [ ] Produtos principais cadastrados com preco e custo coerentes.
- [ ] Estoque inicial registrado ou plano de carga combinado.
- [ ] Cliente e pet de teste cadastrados, quando aplicavel.

## Validacao acompanhada

- [ ] Usuario operacional entrou sem usar uma conta administrativa.
- [ ] Caixa/PDV foi aberto quando exigido pelo fluxo.
- [ ] Venda teste foi concluida.
- [ ] Forma de pagamento foi registrada corretamente.
- [ ] Baixa de estoque foi conferida.
- [ ] Historico e relatorio basico da venda foram conferidos.
- [ ] Modulo fora do Plano Basico nao foi apresentado como parte contratada.
- [ ] Canal oficial de suporte foi testado.

## Aceite operacional

- [ ] Cliente consegue entrar e repetir a jornada principal.
- [ ] Pendencias bloqueantes estao zeradas.
- [ ] Pendencias nao bloqueantes tem responsavel e data.
- [ ] Primeira revisao foi agendada para o dia 1.
- [ ] Revisoes dos dias 3 e 7 foram agendadas.
